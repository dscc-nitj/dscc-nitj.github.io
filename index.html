Welcome to Data Science Code Camp

About DSCC

An intensive four-week Code Camp is being organized for the 1st and 2nd-year students of engineering and technological institutions. The basic motive of this code camp is to provide the basic knowledge of the primary tools of the data science domain and provide hands-on experience to the students at the very initial. This Code camp will provide students with a platform for data science by providing a project-based curriculum.

This course is specially designed according to the requirements of students as well as industry experts. This will include state-of-the-art lectures, interactive and small group-based lab sessions, project-based learning. The present course aims at providing the basic knowledge of R and Python the cutting-edge technologies for Data Science.
